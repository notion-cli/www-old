Notion
the hassle-free Node.js manager

notion help

Notion is a Node.js manager designed to be fast, automatic, and unobtrusive. It features:

  • An ultrafast implementation in Rust;
  • Zero configuration for normal use;
  • Extensibility for custom development environments;
  • Full Yarn support;
  • Automatic, per-project version switching;
  • Support for all shells and all major OSes—including cmd.exe and PowerShell in Windows!

notion install

Notion is still under development. See the project repository for installation instructions.

notion use

If you know how to use Node, you already know how to use Notion. Notion works by making all of your Node executables (including node, npm, yarn, and package binaries) into intelligent shims that switch to the right version for each project. So using the right version of Node for your project is just a matter of typing node—even if that version isn’t yet installed on your machine.

Be sure to read the Getting Started guide to learn more.

notion admin

Do you use Node with a project that needs to customize the development environment? Are you using it at a company that stores its development tools on custom servers? Notion allows complete customization of its pipeline so you can adapt it to your team’s or company’s needs.
